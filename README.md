# NetTool - Automated Network Tool
**NetTool** is a tool to look over your Network.

# Installation 
 
1. git clone https://github.com/disc0nct/NetTool.git
 
1. cd NetTool
 
1. sudo chmod a+x install

1. sudo ./install
 
# Usage

- Just run it from the Terminal by typing 'sudo netTool'.
