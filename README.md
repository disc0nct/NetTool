# NetTool - Automated Network Tool

# Discription

**NetTool** is a tool to look over your Network.

# Installation 
 
1. sudo -i **(grant root access)**

1. git clone https://github.com/disc0nct/NetTool.git
 
1. cd NetTool
 
1. chmod +x install

1. ./install
 
# Usage

- Just run it from the Terminal by typing ' netTool '.
